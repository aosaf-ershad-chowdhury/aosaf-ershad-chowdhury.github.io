The Virtual Aegis

  • Created an application to encrypt and decrypt text files using 5 encryption layers including Caesar Cipher, Vigenere Cipher, Substitution Cipher, and Reverse Cipher.
  • Used the tkinter library in python to create a user interface that improved user experience while using the application

Programming Club Website

[This project is currently in progress]

  • This website will provide interested students with information about my school's programming club. It will also serve as one of the places to update the students regarding any special upcoming programming club events.
  • I will use HTML, CSS, and JavaScript to build the front-end of this website. Bootstrap 5 will also be used to improve the user experience. I later hope to use Django to create an admin site and allow changes to be made easily to the site based on upcoming events.

Web Mining

[This project is currently in progress]

  • An application that will interact with certain websites and parse the retrieved JSON file using Python3.
  • The parsed JSON file will be further processed and stored in an SQLite database. SQL will be used to make necessary changes to the database.
  • Data will later be retrieved from the database and open-source JavaScript codes will be used to visualize the data.
